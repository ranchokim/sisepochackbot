# Bybit Telegram Alert Bot

바이비트 USDT 무기한 선물 종목을 모니터링해 텔레그램으로 알림을 보내는 봇입니다.

## 주요 기능

- **매일 오전 8시 30분(Asia/Seoul) 알림**
  - 24시간 상승률 기준 상위 15개 종목 전송
- **거래량 급증 알림**
  - 주봉(`W`): 현재 거래량이 직전 15개 캔들 평균의 **500% 이상**일 때 알림
  - 4시간봉(`240`), 6시간봉(`360`): 현재 거래량이 직전 10개 캔들 평균의 **800% 이상**일 때 알림
- 동일 캔들에 대한 **중복 알림 방지**

## 파일 구성

- `bybit_alert_bot.py`: 메인 봇 스크립트

## 설치 및 실행

1. Python 3.10+ 환경 준비
2. 의존성 설치

```bash
pip install requests
```

3. `bybit_alert_bot.py` 내부 설정 수정

- `TELEGRAM_TOKEN`
- `CHAT_ID`

4. 실행

```bash
python bybit_alert_bot.py
```

## 주의사항

- 현재 스크립트는 1분 주기로 전체 심볼을 스캔하므로, 네트워크 환경이나 거래소 API 상태에 따라 지연이 발생할 수 있습니다.
- 텔레그램 메시지 길이 제한에 근접할 수 있어, 급증 종목이 매우 많으면 메시지를 분할 전송하는 개선이 필요할 수 있습니다.
